# Vaultwarden

Gestor de contraseñas (compatible Bitwarden).

## Descripcion

Vaultwarden es una implementacion alternativa del servidor Bitwarden.
Usa los clientes oficiales de Bitwarden.

## Acceso

- **Web Vault**: http://vaultwarden.127.0.0.1.traefik.me:9000
- **Admin**: http://vaultwarden.127.0.0.1.traefik.me:9000/admin

## Admin Token

Para acceder al panel de admin, usa el token configurado
en VAULTWARDEN_ADMIN_TOKEN (por defecto: admin123).

## Clientes

Usar clientes oficiales de Bitwarden:
- Extension de navegador
- App movil
- App de escritorio

Configurar servidor: http://vaultwarden.127.0.0.1.traefik.me:9000

## Caracteristicas

- Contraseñas
- Notas seguras
- Tarjetas de credito
- Identidades
- Organizaciones
- Compartir
