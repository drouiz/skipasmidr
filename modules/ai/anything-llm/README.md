# AnythingLLM

Chat con tus documentos usando LLMs locales.

## Descripcion

AnythingLLM permite chatear con documentos usando LLMs
locales como Ollama o APIs externas.

## Acceso

- **URL**: http://anythingllm.127.0.0.1.traefik.me:9000

## Caracteristicas

- Chat con documentos (PDF, TXT, DOCX, etc.)
- Soporte para Ollama (local)
- Multiples workspaces
- Embedding local o via API
- Modo multi-usuario

## Configuracion inicial

1. Acceder a la URL
2. Crear cuenta admin
3. Configurar LLM provider (Ollama recomendado)
4. Crear workspace
5. Subir documentos

## Formatos soportados

- PDF
- TXT
- DOCX
- Markdown
- HTML
- Y mas...
