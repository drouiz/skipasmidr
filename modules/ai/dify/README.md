# Dify

Plataforma de desarrollo de aplicaciones LLM.

## Descripcion

Dify es una plataforma open-source para crear aplicaciones
basadas en LLM con RAG, agentes, y workflows.

## Acceso

- **URL**: http://dify.127.0.0.1.traefik.me:9000

## Dependencias

- PostgreSQL
- Redis

## Caracteristicas

- Creacion de chatbots
- RAG (Retrieval Augmented Generation)
- Agentes AI
- Workflows visuales
- Soporte para multiples LLMs (OpenAI, Anthropic, Ollama, etc.)

## Configuracion

1. Acceder a la URL
2. Crear cuenta admin
3. Configurar proveedor LLM (Ollama para local)
