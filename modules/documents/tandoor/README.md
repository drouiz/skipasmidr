# Tandoor Recipes

Gestor de recetas y planificacion de comidas.

## Descripcion

Tandoor es una aplicacion para gestionar recetas, planificar
comidas y generar listas de compra.

## Acceso

- **URL**: http://recipes.127.0.0.1.traefik.me:9000

## Dependencias

- PostgreSQL

## Caracteristicas

- Gestion de recetas
- Importacion desde URLs
- Planificacion de comidas
- Lista de compra automatica
- Calculadora nutricional
- Multi-usuario

## Configuracion inicial

1. Acceder a la URL
2. Crear cuenta admin
3. Importar o crear recetas
