# Uptime Kuma

Monitor de uptime con interfaz elegante.

## Descripcion

Uptime Kuma es un monitor de uptime self-hosted similar a
Uptime Robot. Interfaz moderna y facil de usar.

## Acceso

- **URL**: http://uptime.127.0.0.1.traefik.me:9000

## Primera configuracion

1. Acceder a la URL
2. Crear usuario admin

## Tipos de monitores

- TCP Port

- Push (heartbeat)
