# Cal.com

Plataforma de programacion de citas open-source.

## Descripcion

Cal.com es una alternativa open-source a Calendly
para programar reuniones y citas.

## Acceso

- **URL**: http://cal.127.0.0.1.traefik.me:9000

## Dependencias

- PostgreSQL

## Caracteristicas

- Paginas de reserva personalizables
- Integracion con Google Calendar, Outlook
- Videoconferencias (Zoom, Google Meet, etc.)
- Workflows y automatizaciones
- Pagos integrados
- API

## Configuracion

1. Acceder a la URL
2. Crear cuenta
3. Configurar disponibilidad
4. Conectar calendarios
