# Wiki.js

Wiki moderno y potente.

## Descripcion

Wiki.js es un wiki open source con editor WYSIWYG/Markdown,
autenticacion, y muchas integraciones.

## Acceso

- **URL**: http://wiki.127.0.0.1.traefik.me:9000

## Primera configuracion

1. Acceder a la URL
2. Completar wizard de instalacion
3. Crear usuario admin

## Dependencias

- PostgreSQL

## Caracteristicas

- Editor visual y Markdown
- Versionado de paginas
- Busqueda full-text
- Diagramas (mermaid, plantuml)
- Autenticacion (OAuth, LDAP, SAML)
- Sincronizacion Git
- Multiidioma
