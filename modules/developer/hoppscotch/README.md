# Hoppscotch

Cliente API open source (alternativa a Postman).

## Descripcion

Hoppscotch es un cliente API rapido y ligero para probar
APIs REST, GraphQL, WebSocket y mas.

## Acceso

- **URL**: http://hoppscotch.127.0.0.1.traefik.me:9000

## Dependencias (opcional)

- PostgreSQL (para sync)

## Variables de entorno

| Variable | Default | Descripcion |
|----------|---------|-------------|
| HOPPSCOTCH_VERSION | latest | Version |
| HOPPSCOTCH_DB | hoppscotch | Base de datos |

## Caracteristicas

- REST API testing
- GraphQL
- WebSocket
- SSE (Server-Sent Events)
- MQTT
- Socket.IO
- Colecciones
- Entornos
- Pre-request scripts
- Historia

## Sin backend

Tambien puedes usar hoppscotch.io directamente en el navegador
sin necesidad de backend.

## Atajos

- Ctrl+Enter: Enviar request
- Ctrl+S: Guardar
- Ctrl+K: Paleta de comandos

## Importar/Exportar

Soporta:
- Postman collections
- OpenAPI/Swagger
- HAR files
