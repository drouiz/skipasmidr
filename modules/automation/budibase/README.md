# Budibase

Plataforma low-code para apps internas.

## Descripcion

Budibase permite crear aplicaciones internas rapidamente
con interfaz drag-and-drop.

## Acceso

- **URL**: http://budibase.127.0.0.1.traefik.me:9000

## Primera configuracion

1. Acceder a la URL
2. Crear cuenta de admin

## Dependencias (opcionales)

- Redis (cache)
- MinIO (storage)

## Variables de entorno

| Variable | Default | Descripcion |
|----------|---------|-------------|
| BUDIBASE_VERSION | latest | Version |

## Caracteristicas

- Builder visual
- Conexion a bases de datos
- APIs REST
- Automatizaciones
- RBAC
- Self-hosted
