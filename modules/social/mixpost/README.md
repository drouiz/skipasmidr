# Mixpost

Gestion de redes sociales self-hosted.

## Descripcion

Mixpost es una alternativa open source a Buffer/Hootsuite para
programar y gestionar publicaciones en redes sociales.

## Acceso

- **URL**: http://mixpost.127.0.0.1.traefik.me:9000

## Dependencias

- PostgreSQL
- Redis

## Variables de entorno

| Variable | Default | Descripcion |
|----------|---------|-------------|
| MIXPOST_VERSION | latest | Version |
| MIXPOST_DB | mixpost | Base de datos |

## Redes soportadas

- Twitter/X
- Facebook
- Instagram
- LinkedIn
- Pinterest
- TikTok
- Mastodon

## Caracteristicas

- Programacion de posts
- Calendario de contenido
- Analytics
- Colaboracion en equipo
- Media library
- AI captions
