# Postiz

Programador de redes sociales open source.

## Descripcion

Postiz es una alternativa open source a Buffer/Hootsuite con AI.

## Acceso

- **URL**: http://postiz.127.0.0.1.traefik.me:9000

## Dependencias

- PostgreSQL
- Redis

## Variables de entorno

| Variable | Default | Descripcion |
|----------|---------|-------------|
| POSTIZ_VERSION | latest | Version |
| POSTIZ_DB | postiz | Base de datos |

## Redes soportadas

- Twitter/X
- LinkedIn
- Facebook
- Instagram
- YouTube
- TikTok
- Pinterest
- Threads
- Mastodon
- Bluesky

## Caracteristicas

- Programacion de posts
- AI para sugerencias
- Multi-cuenta
- Analytics
- Colaboracion
